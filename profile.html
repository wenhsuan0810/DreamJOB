📬 每日 Top 10 職缺 Email

這個網站是純前端、無伺服器,所以每日寄信由 GitHub Actions 排程完成: 機器人每天抓取 104 的職缺(LinkedIn / Indeed 無公開 API,信中會附上你的搜尋深度連結)、 依你的 CV 關鍵字打匹配分數,把前 10 名寄到你的信箱。設定方式:

  1. 按下方按鈕下載 profile.json(只含搜尋條件與 CV 關鍵字,不含履歷全文)。
  2. 把檔案放到 repo 的 profile/profile.json 並 commit。
  3. 在 GitHub repo → Settings → Secrets and variables → Actions 新增: MAIL_USERNAME(Gmail 帳號)、MAIL_PASSWORD(Gmail 應用程式密碼)、 MAIL_TO(收件信箱)。
  4. 完成!每天早上 9 點(台北時間)自動寄出,也可在 Actions 頁面手動觸發 Daily Top 10 Jobs 測試。
